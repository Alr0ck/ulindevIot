# ulindevIot
ESP8266 and Arduino Libraries to Monitoring your Internet of Things Projects.
Simple and easy to use beginners.

//-----------------------------------------------------------------------------//

Wiring between Arduino and Esp8266 :

Arduino ---------- Esp8266
   D3   ----------   tx
   D4   ----------   rx
 3.3V   ----------   3.3V
  GND   ----------   GND
